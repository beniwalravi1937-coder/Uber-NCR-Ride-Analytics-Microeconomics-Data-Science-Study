# Uber NCR Ride Analytics: Microeconomics + Data Science Study

## Overview
This project analyzes ride-hailing demand, pricing, and consumer behavior in Uber NCR using microeconomic theory and data science.

## Dataset
- ~148,000 ride bookings
- Variables: price, distance, vehicle type, time, cancellations, ratings

## Key Concepts Covered
- Two-sided market platform (Uber)
- Demand patterns and peak-load pricing
- Price elasticity of demand
- Surge pricing and equilibrium
- Behavioral economics in ride decisions

## Methodology

### Data Analysis
- Data cleaning and preprocessing
- Feature engineering (time, surge, price/km, etc.)
- Descriptive statistics

### Econometrics
- OLS regression (fare determinants)
- Log-log model (price elasticity)
- Demand estimation

### Machine Learning
- Random Forest
- XGBoost
- Clustering (KMeans)
- PCA

## Key Insights
- Demand shows strong peak-hour patterns (morning & evening)
- Surge pricing reflects demand-supply imbalance
- Driver cancellations driven by low fare & long pickup distance
- Price elasticity varies across vehicle types
- Premium rides contribute disproportionately to revenue
